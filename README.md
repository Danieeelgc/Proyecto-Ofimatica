# 📁 Proyecto: Aplicaciones Ofimáticas - Nexus Gaming Hub

Este repositorio contiene la documentación administrativa y las herramientas de gestión desarrolladas para el centro gaming.

## 🛠️ Contenido del Repositorio
1. **Descripción Corporativa (Word):** Análisis detallado de la empresa, su estructura de empleados e infraestructura informática.
2. **Inventario General (Excel):** Control exhaustivo del hardware y software disponible, organizado por zonas (VIP, Casual y Oficina).
3. **Gestión de Incidencias (Excel):** Sistema de registro con formato condicional y validación de datos para el seguimiento técnico.
4. **Presentación Ejecutiva (PowerPoint):** Resumen visual del sistema informático diseñado.

## 🚀 Competencias Aplicadas
- **Procesamiento de texto:** Uso de estilos, índices automáticos y SmartArt.
- **Hojas de cálculo:** Aplicación de filtros, formato condicional y menús desplegables.
- **Diseño de presentaciones:** Estructura visual profesional y síntesis de información.
